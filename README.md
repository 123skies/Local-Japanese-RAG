# Local-Japanese-RAG
High-precision local RAG system optimized for Japanese documents. Features Hybrid Search (BM25 + Vector), Reranking (Cross-Encoder), and strict citation mapping. Built with Streamlit, Ollama, LanceDB, and SudachiPy. (ScholarScope) / 日本語文書に特化した高精度なローカル完結型RAG。ハイブリッド検索、リランキング、和暦対応、出典明示機能を搭載。
